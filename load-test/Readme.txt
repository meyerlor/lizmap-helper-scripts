This script will put a heavy load on your server (basically "DDOS"). It uses selenium (so you need to have google chrome installed) to simulate 
users accessing your Lizmap map. Change the URL and the number of concurrent users in the very bottom of the script (in the main function!).

Launch from your PC, NOT from your server.

Optionally, launch the collect-logs script on your targeted server before starting the load test so you record meaningful results :)
