This script will download files from a QFieldCloud account into a specified
folder on your server. I recommend to run this script as a cronjob.

You can filter which files you want to download. See more on https://pypi.org/project/qfieldcloud-sdk/

Upload from your Lizmap media folder into QFieldCloud also works, you need to adjust the script accordingly.

This creates real seamless fieldwork, especially if you have your data in a postgres database.
