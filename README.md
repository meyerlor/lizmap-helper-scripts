# lizmap-helper-scripts
Collection of my helper scripts for various tasks i encountered using the awesome Lizmap Web Clien (LWC)

It contains:
1. A script to batch create users from a csv
2. A log collector for anything remotely connected to the performance of your LWC software stack (deprecated, use the much nicer [Monitoring Tool with a dedicated Repository](https://github.com/meyerlor/qgis-server-monitoring)
3. A script which simulates a lot of concurrent users on your LWC (perfect in combination with the log collector to spot problems in your server's configuration)
4. An "automatic" Lizmap Updater (works also for downgrades)
5. A file transfer script for media files recorded with Qfield and saved in QFieldCloud
6. Very simplified, easy to replicate installation guide for py-qgis-server (as used at FOSSGIS 2026 with Peters Server ;) )
